## For Backend

1. masuk ke folder data `cd data/`
2. jalankan `npm install`
3. jalankan `npx json-server data.json`

## For Frontend

1. masuk ke folder my-app `cd my-app/`
2. jalankan `npm install`
3. jalankan `npm run dev`
